LinearTween

Un Tween es una interpolacion entre 2 puntos. La clase puede interpolar 2 puntos que se le pasen, o puede ser asignada a un Sprite para que interpole todos sus movimientos

(Object p1, Object p2)

public LinearTween play

Habilita el avance del puntero interno... o sea... play, obvio. Inhabilita el avance del puntero interno... o sea... pause, obvio.

public Object requestFrame

Obtiene un frame de lo interpolado con _fnTween o tweenTo. Si llega al final de la interpolacion dispara el evento onFinish.

public LinearTween reset

Inicializa el Tween, vacia _frames, pone _framePointer en 0, y algo mas que no recuerdo.

public LinearTween resetPointer

(optional int frame)

Inicializa el puntero interno, si se le proporciona un frame lo inicializa en ese punto.

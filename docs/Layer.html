Layer

Un Layer es un contenedor de Sprites. Un Layer puede tener un ancho/alto superior al del canvas que lo contiene, y puede desplazarse con un scroll. Tambien se le puede asignar una imagen que actue como limite para los sprites (boundary), y se le puede asignar un background.
